# W.I.P.

## Introduction

Fortify Bootstrap is a combination of [Laravel Fortify](https://github,com/laravel/fortify) and [Bootstrap 4](https://getbootstrap.com). 

## Installation

```
composer require akukoder/fortify-bootstrap
```
